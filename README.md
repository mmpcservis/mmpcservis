# MM PC Servis Website

Moderan single-page website za **MM PC Servis** optimiziran za **GitHub Pages**.

## Što je unutra

- `index.html` — kompletan website
- `.nojekyll` — osigurava da GitHub Pages servira datoteke bez Jekyll obrade

## Kako objaviti na GitHub Pages

1. Napravi novi GitHub repo, npr. `mm-pc-servis`
2. Uploadaj `index.html` i `.nojekyll`
3. Otvori **Settings > Pages**
4. Pod **Build and deployment** odaberi:
   - **Source:** Deploy from a branch
   - **Branch:** `main` / root
5. Spremi
6. Nakon kratkog vremena stranica će biti online na adresi:

`https://USERNAME.github.io/REPOSITORY/`

## Obavezno zamijeni ovo u `index.html`

Pronađi ovaj dio u JSON-LD sekciji:

`https://USERNAME.github.io/REPOSITORY/`

i zamijeni ga svojom pravom GitHub Pages adresom.
