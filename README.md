# MM PC Servis Website

Jednostavan i brz landing page za **MM PC Servis (Čitluk / Mostar)**.

Website je napravljen kao **single HTML file** kako bi bio jednostavan za:
- GitHub Pages hosting
- Netlify hosting
- lokalni SEO
- brzu izradu bez troškova

Cilj stranice je **generirati telefonske pozive i upite za PC servis**.

# Kako pokrenuti website

Jednostavno otvorite HTML file u browseru:

`mm_pc_servis_landing_page.html`

Website ne zahtijeva:
- server
- bazu podataka
- instalaciju

---

# Besplatan hosting

Website se može besplatno hostati na:

## GitHub Pages

1. upload HTML file u repo
2. otvorite repository settings
3. enable **GitHub Pages**
4. website će biti live

`https://mmpcservis.github.io/repository`

---

## Netlify (najlakše)

1. otvorite Netlify Drop
2. povucite HTML file
3. website je odmah online

---

## Vercel

1. napravite GitHub repo
2. import repo u Vercel
3. website se automatski deploya
